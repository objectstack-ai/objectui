The Span component is an inline container. **It is now deprecated in favor of semantic Shadcn-based components** that provide better accessibility and design system integration.

## Migration Guide

Instead of using `span`, use these Shadcn alternatives:

### For Badges → Use `badge`

<DemoGrid>
  <ComponentDemo
    schema={{
      type: 'badge',
      variant: 'default',
      children: [{ type: 'text', content: 'Badge Style' }]
    }}
    title="Default Badge"
  />
  <ComponentDemo
    schema={{
      type: 'badge',
      variant: 'secondary',
      children: [{ type: 'text', content: 'Secondary' }]
    }}
    title="Secondary Badge"
  />
</DemoGrid>

### For Status Indicators → Use `badge` with variants

<ComponentDemo
  schema={{
    type: 'flex',
    gap: 2,
    children: [
      { type: 'badge', variant: 'default', children: [{ type: 'text', content: '● Active' }] },
      { type: 'badge', variant: 'secondary', children: [{ type: 'text', content: '● Pending' }] },
      { type: 'badge', variant: 'destructive', children: [{ type: 'text', content: '● Inactive' }] }
    ]
  }}
  title="Status Badges"
/>

### For Inline Text → Use `text` with className

<ComponentDemo
  schema={{
    type: 'text',
    content: 'The quick brown fox jumps over the lazy dog.',
    className: 'text-base'
  }}
  title="Text Component"
/>
