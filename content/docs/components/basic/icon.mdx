## Schema

```plaintext
interface IconSchema {
  type: 'icon';
  name: string;              // Lucide icon name (kebab-case)
  size?: number | 'sm' | 'md' | 'lg' | 'xl';
  color?: string;            // Tailwind color class
  className?: string;
}
```
