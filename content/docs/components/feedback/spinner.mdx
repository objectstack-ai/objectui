The Spinner component displays a loading indicator.

## Basic Usage

<ComponentDemo
  schema={{
    type: 'spinner'
  }}
  title="Default Spinner"
/>

## Schema

```plaintext
interface SpinnerSchema {
  type: 'spinner';
  size?: 'sm' | 'md' | 'lg';
  className?: string;
}
```

## Examples

### In Button

<ComponentDemo
  schema={{
    type: 'button',
    label: 'Loading...',
    disabled: true,
    children: [
      { type: 'spinner', size: 'sm' },

/>
