The Kbd component displays keyboard shortcuts in a styled format.

## Basic Usage

<ComponentDemo
  schema={{
    type: 'kbd',
    keys: ['Ctrl', 'C']
  }}
  title="Copy Shortcut"
/>

## Schema

```plaintext
interface KbdSchema {
  type: 'kbd';
  keys: string[];                 // Keyboard keys
  className?: string;
}
```
